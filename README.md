# 🍒 Cherry

A personal life management Android app built with Kotlin and Jetpack Compose.

## 📱 Features
- **Dashboard** — Home screen with quick access to all features
- **Attendance Tracker** — Track and predict your attendance
- **Password Manager** — Secure local password vault
- **Advice System** — Store and search advice from seniors & professors

## 🛠️ Tech Stack
- Language: Kotlin
- UI: Jetpack Compose
- Architecture: MVVM
- Database: Room (coming soon)
- Security: Android Keystore + AES (coming soon)
